Three Levels of Engineering

Concentric layers — each level encompasses the previous.

Harness Engineering Outermost

Tool orchestration · state persistence · error recovery · verification · safety · lifecycle

Context Engineering Middle

Manages what the model sees and when. 30%+ degradation when key content falls in mid-window.

Prompt Engineering Innermost

Crafts the instructions the model receives. Foundation, but alone insufficient for production.
